#  Chifoumi (Pierre-Feuille-Ciseaux)


> **Voir le jeu en ligne :** [Cliquez ici pour jouer](https://nutella9775.github.io/chifoumi/) ##

Bienvenue sur mon adaptation du célèbre jeu **Pierre-Feuille-Ciseaux**.
Ce projet a été réalisé pour mettre en pratique les fondamentaux du **JavaScript** et l'interaction avec le **DOM** (Document Object Model).

Le joueur affronte l'ordinateur qui effectue un choix aléatoire à chaque tour. Le premier arrivé à un certain score (ou le score infini selon votre version) gagne la gloire !

## Technologies Utilisées

| Technologie | Usage |
| :--- | :--- |
| **HTML5** | Structure sémantique de la page. |
| **CSS3** | Mise en forme. |
| **JavaScript (ES6)** | Logique du jeu, gestion des événements (`click`) et manipulation du DOM. |
